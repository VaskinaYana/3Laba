# Задание 1

### Текст задачи

Решить задачу из лабораторной работы №2 (Задание 1. List.map) для последовательности.
Задача:На основе списка, содержащего цифры от 1 до 9, получить список из их римских
обозначений.


### Алгоритм решения

1. Запрашиваем у пользователя числа от 1 до 9 с сообщением о завершении ввода
2. Создаем рекурсивную функцию readNumbers, которая возвращает последовательность (seq):
   - Считываем введенную строку с консоли
   - Пытаемся преобразовать строку в число с помощью System.Int32.TryParse
   - Если преобразование успешно и число находится в диапазоне от 1 до 9, возвращаем это число и рекурсивно продолжаем ввод
   - Если ввод некорректен (не число или число вне диапазона), завершаем последовательность
3. Вызываем функцию readNumbers и сохраняем результат в переменную numbers как последовательность
4. Создаем функцию Roman для преобразования чисел в римские цифры:
   - Сопоставляем каждое число от 1 до 9 с соответствующим римским символом
   - Для остальных значений возвращаем пустую строку
5. Применяем функцию Roman к каждому элементу последовательности с помощью Seq.map, получая последовательность римских чисел
6. Преобразуем полученную последовательность в список с помощью Seq.toList и выводим результат на экран
7. Возвращаем 0 как код успешного завершения программы

# Задание 2

### Текст задачи

Решить задачу из лабораторной работы №2 (Задание 2. List.fold) для последовательности.
Задача:Список содержит десятичные цифры. Составить из них число.

### Алгоритм решения

1. Запрашиваем у пользователя десятичные цифры с сообщением о завершении ввода
2. Создаем рекурсивную функцию readDigits, которая возвращает последовательность (seq):
   - Считываем введенную строку с консоли
   - Пытаемся преобразовать строку в число с помощью System.Int32.TryParse
   - Если преобразование успешно и число находится в диапазоне от 0 до 9, возвращаем эту цифру и рекурсивно продолжаем ввод
   - Если ввод некорректен (не число или число вне диапазона), завершаем последовательность
3. Вызываем функцию readDigits и сохраняем результат в переменную digits как последовательность
4. Формируем число из последовательности цифр:
   - Используем Seq.fold для последовательной обработки каждой цифры
   - Начальное значение аккумулятора равно 0
   - Для каждой новой цифры умножаем текущее значение аккумулятора на 10 и прибавляем цифру
   - Таким образом, цифры выстраиваются в правильном порядке (например, [1;2;3] превращается в 123)
5. Выводим полученное число на экран
6. Возвращаем 0 как код успешного завершения программы
